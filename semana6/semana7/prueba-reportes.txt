Al enviar una petición POST a /reportes con el siguiente JSON:
{
  "tipo": "Infraestructura",
  "descripcion": "Daño en alumbrado público"
}

El servidor responde con:
{
  "mensaje": "Reporte registrado",
  "reporte": {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Daño en alumbrado público"
  }
}

Al consultar GET /reportes, el servidor devuelve la lista de todos
los reportes registrados durante la sesión actual del servidor.
Si el servidor se reinicia, la lista vuelve a estar vacía porque
los datos se guardan en memoria y no en una base de datos.
